# TableauChallenge
*self education, acquire basic skills*

##### 01.Barchart 
- cтолбиковый график: использовать для точного сравнения значений между категориями.
##### 02.Linechart 
- линейный график: показывает тренды, сезонность и изменения во времени.
##### 03.Big Number 
- большие числа: использовать для быстрого понимания текущего состояния метрики.
##### 04. Tables 
- таблицы: для проверки и детализации, а не для поиска трендов.
##### 05. Pie, Donut 
- круг, кольцо: доля от целого использовать только при небольшом количестве категорий;
##### Sunburst
- солнечные лучи: иерархия долей, показывает структуру: категория → подкатегория.
##### 06. Treemap 
- дерево: много категорий важен вклад, хорошо показывает, кто реально влияет на результат.
##### 07. Gantt Chart 
- диаграмма Гантта: задачи и сроки, использовать для контроля выполнения задач и выявления рисков по срокам.
